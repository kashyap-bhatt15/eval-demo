LangChain LLM Evaluation Interface

Test your LLM responses against expected outputs using LangChain evaluators.

Input Test Cases (JSON)

Example Format:
{
  "test_cases": [
    {
      "prompt": "Hello, how are you?",
      "expected_output": "Hello! I'm doing well, thank you for asking. How can I help you today?",
      "test_name": "greeting"
    },
    {
      "prompt": "What is the capital of France?",
      "expected_output": "The capital of France is Paris.",
      "test_name": "factual_qa"
    },
    {
      "prompt": "Write a short poem about the ocean.",
      "expected_output": "Waves crash upon the sandy shore,\nBlue depths hold secrets and much more.\nThe ocean vast, both calm and wild,\nNature's beauty, unreconciled.",
      "test_name": "creative_writing"
    }
  ]
}
Enter test cases with prompts and expected outputs.
